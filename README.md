# 📱 Qutbah Sheikh Mohamed

An Android application for listening to inspiring Islamic khutbahs by Sheikh Mohamed.

## Features

- 🎧 High-quality khutbah audio
- 📂 Organized audio library
- ▶️ Easy-to-use audio player
- 📱 Clean and simple interface
- ⚡ Fast and lightweight

## Download

**Latest APK:**

https://github.com/mwalimu-abdi/Qutbah-Sheikh-Mohamed/releases/latest

## Installation

1. Download the latest APK.
2. Open the downloaded file.
3. If prompted, allow installation from unknown sources.
4. Install the app.
5. Open and enjoy listening.
